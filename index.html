Study Flashcards

What is HTML?
A markup language used to structure web pages.
What does CSS do?
It styles and designs web pages.
What is a div?
A container element used to group content.
What is a transition?
A smooth change between CSS property values.
What is an animation?
A sequence of keyframed style changes.
What does hover mean?
When your mouse is placed over an element.
What is a grid layout?
A CSS system for creating rows and columns.
What is opacity?
Controls how transparent or visible something is.
